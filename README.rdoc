== README

The pb-form application was created at AMIA Hack Day 2014.

To run this code on Heroku:

Create a heroku account and confirm that you can connect.
Add the heroku gem to your local machine with "bash gem install heroku""
Create a heroku app with "heroku apps:create"
Migrate the database with "heroku run rake db:migrate"
Restart the workers with "heroku apps:restart"
Open your new Heroku app with "heroku apps:open"
